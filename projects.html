Project: Smart Blood Net

Role: Developer

Description:

Smart Blood Net is a web-based emergency response platform designed to connect blood donors with hospitals in real time. The system enables rapid identification of available donors and facilitates immediate communication during critical situations.

The platform improves accessibility to life-saving resources by reducing delays in donor matching and enhancing coordination between medical institutions and volunteers.

Key Features

  • Real-time donor search based on location and blood group
  • Emergency notification system
  • Hospital and donor registration modules
  • Secure data handling

Technologies Used

HTML, CSS, JavaScript, Database (MySQL)
